---
## APR (Annual Percentage Rate)

```
APR = (Reward / Stake) * 365 * 100
```

Reward: Money will get in year

Stake: Money user deposit

## APY (Annual Percentage Yield)

```
APY = (1 + (Reward / Stake)) ^ 365 - 1
```

Reward: Money will get in year

Stake: Money user deposit
