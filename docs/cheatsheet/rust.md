---
title: Setup Rust
---

## Install

`curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh`

- Check version

`rustc --version`

- Run program

`rustc main.rs`

## Cargo

- Check version

`cargo --version`

- Create project

`cargo new cargo-projects`

- Run program

```bash
cargo run
cargo run --bin 1-function
cargo build
cargo check
```
